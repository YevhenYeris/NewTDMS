
# NewTDMS

Даний проєкт є реалізацією системи управління табличними базами даних (СУТБД).

СУТБД підтримує роботи з такими типами даних:
- integer
- real
- char
- string
- **$**
- **$Invl**.

Реалізовано додаткову варіантну операцію - **перейменування стовпчиків бази даних**.
